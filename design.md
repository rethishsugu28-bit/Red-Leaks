# Red Leaks - Design Style Guide

## Design Philosophy

### Core Values
- **Empowerment**: Design that makes women feel confident and in control
- **Trust**: Professional, reliable aesthetic that builds confidence in sensitive health data
- **Accessibility**: Inclusive design that works for all women regardless of technical ability
- **Innovation**: Modern, forward-thinking visual language that reflects cutting-edge technology

### Visual Language
- **Feminine Strength**: Bold yet gentle, combining soft curves with confident typography
- **Health Tech**: Clean, medical-grade precision meets warm, human-centered design
- **Premium Care**: High-quality aesthetic that reflects the value of women's health
- **Community Focus**: Design that connects individual experience to collective empowerment

## Color Palette

### Primary Colors
- **Coral Rose**: #FF6B6B - Primary brand color, warm and approachable
- **Deep Berry**: #C44569 - Secondary accent, sophisticated and trustworthy
- **Soft Blush**: #FFF5F5 - Background tint, gentle and calming

### Supporting Colors
- **Sage Green**: #95C7A3 - Success states, health indicators
- **Warm Gray**: #6C757D - Neutral text, subtle elements
- **Pure White**: #FFFFFF - Clean backgrounds, card surfaces

### Accent Colors
- **Golden Honey**: #FFC107 - Highlights, call-to-action elements
- **Lavender Mist**: #E8DAFF - Special features, premium options

## Typography

### Primary Font: "Inter" (Sans-serif)
- **Headings**: Inter Bold (700) - Clean, modern, highly readable
- **Body Text**: Inter Regular (400) - Excellent for digital interfaces
- **UI Elements**: Inter Medium (500) - Perfect for buttons and labels

### Display Font: "Playfair Display" (Serif)
- **Hero Headlines**: Playfair Display Bold - Elegant, editorial feel
- **Section Titles**: Playfair Display SemiBold - Sophisticated accent

### Font Hierarchy
- **H1 (Hero)**: 3.5rem / 56px - Playfair Display Bold
- **H2 (Section)**: 2.5rem / 40px - Inter Bold
- **H3 (Subsection)**: 1.75rem / 28px - Inter SemiBold
- **Body**: 1rem / 16px - Inter Regular
- **Small**: 0.875rem / 14px - Inter Regular

## Visual Effects & Animation

### Background Effects
- **Aurora Gradient Flow**: Subtle animated gradient using coral and blush tones
- **Floating Particles**: Gentle, organic shapes representing care and support
- **Liquid Displacement**: Smooth, flowing animations suggesting comfort and flow

### Text Effects
- **Color Cycling Emphasis**: Key phrases pulse through coral to berry tones
- **Gradient Text Animation**: Hero headlines with animated coral-to-berry gradient
- **Split-by-Letter Stagger**: Reveal animations for important messaging

### Interactive Effects
- **3D Tilt**: Product cards lift and tilt on hover with soft shadows
- **Glow Transitions**: Buttons emit warm coral glow on interaction
- **Morphing Shapes**: UI elements smoothly transform between states

## Layout & Spacing

### Grid System
- **Desktop**: 12-column grid with 24px gutters
- **Tablet**: 8-column grid with 20px gutters  
- **Mobile**: 4-column grid with 16px gutters

### Spacing Scale
- **XS**: 4px - Fine details
- **SM**: 8px - Component spacing
- **MD**: 16px - Section spacing
- **LG**: 32px - Major sections
- **XL**: 64px - Page sections

### Component Styling
- **Border Radius**: 12px for cards, 8px for buttons, 6px for inputs
- **Shadows**: Soft, warm shadows using coral tint (0 4px 20px rgba(255,107,107,0.1))
- **Borders**: 1px solid with subtle coral tint for focus states

## Imagery Style

### Photography Direction
- **Authentic Women**: Real, diverse women in natural, comfortable settings
- **Soft Lighting**: Warm, golden hour lighting that feels intimate and safe
- **Lifestyle Focus**: Women going about daily life with confidence and ease
- **Product Integration**: Sanitary products shown naturally, without shame

### Illustration Style
- **Organic Shapes**: Flowing, curved illustrations that mirror natural cycles
- **Minimalist Approach**: Clean, simple illustrations focusing on essential elements
- **Warm Color Palette**: Using brand colors in soft, approachable ways
- **Abstract Concepts**: Visual metaphors for tracking, prediction, and care

### Icon Design
- **Rounded Corners**: All icons use 2px border radius for softness
- **Consistent Weight**: 2px stroke weight for all line icons
- **Brand Integration**: Subtle coral accents on key functional icons
- **Accessibility**: All icons paired with text labels for clarity

## Responsive Design

### Breakpoints
- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Desktop**: 1024px - 1440px
- **Large**: 1440px+

### Mobile Adaptations
- **Touch Targets**: Minimum 44px for all interactive elements
- **Typography**: Slightly larger base font size (18px) for readability
- **Spacing**: Increased padding for comfortable thumb navigation
- **Simplified Layouts**: Single-column layouts with clear visual hierarchy

## Accessibility Standards

### Color Contrast
- **Text on White**: Minimum 4.5:1 contrast ratio
- **Interactive Elements**: Minimum 3:1 contrast ratio
- **Focus States**: High contrast outline using brand colors

### Motion & Animation
- **Reduced Motion**: Respect user preferences for reduced motion
- **Essential Animations**: Only use animation to enhance understanding
- **Performance**: Smooth 60fps animations with hardware acceleration
